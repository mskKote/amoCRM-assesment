## Задание 1

Напишите реализацию таймера.

- шаг анимации таймера 1 секунда;
- форматирование таймера – “hh:mm:ss”;
- в инпут можно вводить только числа.

Например 01:12:59 - один час, 12 минут, 59 секунд.
